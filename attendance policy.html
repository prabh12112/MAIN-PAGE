Attendance Policy

Online Submission Guidelines

Comprehensive rules and regulations for the online attendance system used during official class hours at D.A.V. College Jalandhar. Please read carefully to ensure compliance.

1. Overview

This policy outlines the rules and regulations regarding the online attendance system used during official class hours at D.A.V. College Jalandhar. The system ensures fairness, authenticity, and accuracy in collecting student attendance through a secure, location-based web portal.

2. Attendance Window

  • Attendance must be marked strictly within the designated time slots announced for each session.
  • Example valid slots: 10:30 AM â€“ 11:00 AM or 12:00 PM â€“ 12:30 PM.
  • Attendance outside the time window will be rejected automatically.

3. Location Verification

  • Students must be within a Specified radius of the Classroom.
  • Location is verified using the Google Geolocation API.
  • Manual inputs will not be accepted.

4. Single Submission Policy

  • Only one attendance submission is allowed per student per day.
  • IP address logging prevents multiple submissions.

5. Submission Process

  1. Access the attendance portal via the official class link.
  2. Allow location access when prompted.
  3. Click the transparent blue 'Continue' button if within the allowed location and time.
  4. Fill the linked Google Form.

6. Technical Integrity

  • All data is stored securely in Google Sheets.
  • The frontend is hosted on Web Pages.
  • Realtime validations prevent unauthorized entries.

7. Violations and Disciplinary Action

The following actions may result in disciplinary consequences, including nullification of attendance:

  • Submitting attendance on behalf of others.
  • Manipulating the system or bypassing verification.
